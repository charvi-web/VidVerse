# NWealth Capital — Wealth Management Platform

A complete web application for **NWealth Capital**, a wealth-management firm, built with
**Next.js 14 (App Router)**, **TypeScript**, **Tailwind CSS v4** and **AWS Amplify Gen 2**
(Cognito auth + AppSync/DynamoDB data). It covers the full client journey — from discovering
services, to booking a consultation, authenticating, onboarding, and managing everything from
a client portal, plus an internal admin portal.

## Architecture

- **UI system** — `app/globals.css` uses the default shadcn (luma preset) neutral theme + a
  local component library in `components/ui`, `components/site`, `components/marketing`,
  `components/portal`.
- **Content/data** — `lib/services.ts`, `lib/site.ts`, `lib/mock.ts` (portal/admin sample data).
- **Backend** — `amplify/auth/resource.ts` (email login, `ADMIN`/`ADVISOR` groups) and
  `amplify/data/resource.ts` (Booking, Enquiry, Lead, ClientProfile, Consultation,
  ServiceRequest, Document, Invoice, Message models with owner/group authorization).
- Auth uses the AWS Amplify Authenticator; portal/admin screens use sample data for the
  interactive wireframe/prototype.

## Getting started

```bash
npm install
npx ampx sandbox        # provisions Cognito + AppSync and writes amplify_outputs.json
npm run dev             # http://localhost:3000
```

`amplify_outputs.json` is generated by the Amplify backend (sandbox or Amplify Hosting) and is
git-ignored. The UI renders locally without a deployed backend; the sign-in flow requires a
real backend.

```bash
npm run build           # production build (all 37 routes prerender)
```

## Deploying to AWS

See the [Amplify deployment guide](https://docs.amplify.aws/nextjs/start/quickstart/nextjs-app-router-client-components/#deploy-a-fullstack-app-to-aws).
`amplify.yml` runs `ampx pipeline-deploy` for the backend and `next build` for the frontend.
